## <a id="html">HTML 작성 규칙</a>

- p 태그는 단락에만 사용

- list안에는 최대한 semantic 의미를 갖는 태그는 사용을 자제할 것

- list에서 텍스트 강조, 넘버링 태그 외에 태그 엘리먼트와 텍스트 엘리먼트를 바로 나열하지 말것
